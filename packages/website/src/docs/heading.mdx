## Supported props

| Name     | Type        | Default | Description               |
| :------- | :---------- | :-----: | :------------------------ |
| size     | System prop |    3    | mapped to fontSizes scale |
| weight   | boolean     |  false  | font weight               |
| truncate | boolean     |  false  | truncate text             |
| color    | System prop | grays.8 | text color                |
