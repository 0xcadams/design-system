### Custom props

| Name       | Type                   | Default | Description           |
| :--------- | :--------------------- | :-----: | :-------------------- |
| `size`     | ResponsiveValue<0...5> |   `3`   | Changes the font size |
| `bold`     | Boolean                | `false` | Makes the text bold   |
| `truncate` | boolean                | `false` | Truncates the text    |
