Radix is a Component Library built on top of [Radix System](/docs/radix-system).

### Installing Radix

Radix is currently under heavy development, but can install it by running either of these commands:

```js
npm install @modulz/radix
```

or

```js
yarn add @modulz/radix
```

### RadixProvider

### Importing components from Radix

All components can be imported directly.

```js

```
