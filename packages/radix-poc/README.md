### Develop

#### Install dependencies

```
yarn
```

#### Run Storybook

```
yarn storybook
```

#### Watch compiled lib

```
yarn watch
```

> Useful when developing in `link` mode

#### Lint

```
yarn lint
```

#### Prettier

```
yarn format
```

---

### Build & Release

### Build

```
yarn build
```

#### Release

```
npm run release
```

> For some reason this doesn't work with `yarn`
